/** * INSTRUÇÕES DE INSTALAÇÃO (fazer uma única vez): * * 1. Abra a planilha no navegador. * 2. Menu Extensões (Extensions) → Apps Script. * 3. Apague o conteúdo padrão e cole este arquivo inteiro. * 4. Clique no ícone de salvar (disquete). * 5. No menu de funções (dropdown no topo, ao lado de "Depurar"), selecione "syncPainelData" * e clique em "Executar" (▶). Na primeira vez, o Google vai pedir autorização: * clique em "Revisar permissões" → escolha sua conta → "Avançado" → "Acessar [nome da planilha] (não seguro)" * → "Permitir". Isso é normal, é só o Google avisando que o script vai editar a própria planilha. * 6. Depois de rodar uma vez com sucesso (sem erro em vermelho), vá no ícone de relógio * (Acionadores / Triggers) no menu lateral esquerdo → "+ Adicionar acionador". * 7. Configure: * Função a executar: syncPainelData * Origem do evento: Do Google Sheets * Tipo do evento: Ao editar (crie um segundo acionador igual, mas com "Ao mudar" / On change) * 8. Salvar. * * A partir daqui, toda vez que você editar qualquer aba OU criar uma aba nova, * uma aba oculta chamada "PAINEL_DATA" é atualizada automaticamente com o conteúdo * da aba mais recente (a mais à direita). O painel sempre lê essa aba fixa, * então você pode continuar criando uma aba nova por mês sem quebrar nada. */ function syncPainelData() { const ss = SpreadsheetApp.getActiveSpreadsheet(); const allSheets = ss.getSheets(); // Ignora a própria aba de destino ao procurar "a mais recente" const candidateSheets = allSheets.filter(s => s.getName() !== 'PAINEL_DATA'); if (candidateSheets.length === 0) return; const latest = candidateSheets[candidateSheets.length - 1]; // aba mais à direita const range = latest.getDataRange(); const values = range.getValues(); let painel = ss.getSheetByName('PAINEL_DATA'); if (!painel) { painel = ss.insertSheet('PAINEL_DATA'); } painel.clearContents(); if (values.length > 0) { painel.getRange(1, 1, values.length, values[0].length).setValues(values); } // Mantém a aba oculta pra não atrapalhar seu dia a dia painel.hideSheet(); } function onOpen() { syncPainelData(); }
